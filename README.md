### 🤖 [Revitalização de Rover Didático](https://github.com/Rover-Project/Rover)

* **Tecnologias:** ![Raspberry Pi](https://img.shields.io/badge/Raspberry%20Pi-C51A4A?style=flat-square&logo=raspberry-pi&logoColor=white)
